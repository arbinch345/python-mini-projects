<h3># Python Mini Projects</h3>

<p>
    This repository contains beginner-level Python projects that I built while learning Python. These projects helped me practice programming fundamentals, problem-solving, functions, exception handling, loops, and modules.
</p>

<p>

    1. Calculator 
    A simple calculator built using functions that can perform basic arithmetic operations. It also includes:

    A fun game where the user tries to guess a randomly generated number.

    - Provides feedback on guesses 
    - Improves logical thinking 

    # 3. Rock Paper Scissors Game 
    A classic game played against the computer. 
    - Uses the `random` module 
    - Takes user input 
    - Determines the winner based on game rules 

    # Purpose 
    The purpose of these projects is to: 
    - Strengthen Python fundamentals 
    - Apply theoretical concepts in practical programs 
    - Improve coding and debugging skills
