<h3>Python Intermediate Projects</h3>

<p>A collection of intermediate-level Python CLI projects built to practice core Python concepts such as functions, exception handling, file handling, modules, game logic, and JSON data storage.</p>
<br>

<p>
    # Projects I have made: 

    1. Banking Simulation System 
    A command-line banking application where users can:
    - Check account balance
    - Deposit money
    - Withdraw money

    Features:
    - Prevents depositing invalid amounts like 0
    - Prevents withdrawing more money than available balance
    - Uses try...except for error handling
    - Function-based structure
    - Simple CLI interface

    2. Password Strength Checker & Generator 
    A CLI-based password utility that can:
    - Check password strength
    - Detect common passwords
    - Generate strong random passwords

    Features:
    - Uses a custom list of common passwords
    - Warns users if the password is too common
    - Generates passwords based on user-defined length

    3. Guess The Deck Game 
    A fun command-line card guessing game where players predict:
    - Higher card
    - Lower card
    - Red card
    - Black card

    Players earn points for correct guesses and lose lives for wrong answers.

    Features:
    - Full deck generation system
    - Card shuffling
    - Score and streak system
    - Bonus points for winning streaks
    - High score saving using JSON
    - Color detection (Red/Black)
    - Persistent high score storage
