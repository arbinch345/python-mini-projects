<h3> # Python Mini Projects 🚀 </h3>

<p>

<p>

    The projects focus on:

    * Python fundamentals
    * Problem-solving
    * CLI-based applications
    * Functions and modular code
    * Exception handling
    * File handling
    * Game logic
    * Python standard libraries
</p>
